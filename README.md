# 逐风少年

一款使用 Phaser 3、TypeScript 和 Vite 制作的横版像素风自行车逃生游戏。小男孩需要穿过台风登陆前的沿海小镇，跨越障碍并在风墙追上前抵达避风站。

## 游戏内容

- 自动骑行与冲刺体力系统
- 跳跃和二段连跳
- 六种道路障碍与可收集风车
- 动态台风追逐、暴雨和闪电
- 桌面键鼠与移动端触控操作
- 自适应桌面和手机画面

## 本地运行

```bash
npm install
npm run dev
```

访问 `http://127.0.0.1:5173/`。

## 构建

```bash
npm run build
```

生产文件会输出到 `dist/`。

## 技术栈

- Phaser 3
- TypeScript
- Vite
